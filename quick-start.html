安装软件

AI Prompt Studio 目前支持 Windows 系统，macOS 和 Linux 版本正在开发中。

系统要求

  • Windows 10 或更高版本
  • 4GB RAM 或更高
  • 500MB 可用磁盘空间
  • 1024x768 分辨率或更高

安装步骤

  1. 从官网下载 Windows 版本的安装包
  2. 双击运行安装程序
  3. 按照安装向导完成安装
  4. 安装完成后，从开始菜单启动 AI Prompt Studio

首次运行

首次启动 AI Prompt Studio 时，系统会进行初始化设置。

初始化步骤

  1. 选择数据存储位置
  2. 设置界面主题（明亮/暗黑）
  3. 配置基本偏好设置
  4. 完成初始化，进入主界面

基本使用

了解 AI Prompt Studio 的基本功能和操作方法。

创建提示词

  1. 点击"新建"按钮
  2. 选择提示词类型（文本/图片/视频）
  3. 输入提示词内容
  4. 添加标签和分类
  5. 保存提示词

搜索提示词

  • 使用顶部搜索框进行全文搜索
  • 通过标签筛选相关内容
  • 按类型过滤提示词
  • 使用高级搜索功能

使用技巧

一些实用的使用技巧，帮助您更高效地使用 AI Prompt Studio。

快捷键

  • Ctrl + N - 新建提示词
  • Ctrl + F - 搜索
  • Ctrl + S - 保存
  • F11 - 全屏模式

最佳实践

  • 为提示词添加描述性标签
  • 定期备份数据
  • 使用分类管理相关提示词
  • 及时清理不需要的内容
